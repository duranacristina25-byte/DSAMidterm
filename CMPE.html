Visual studio code set up
1. Go to the official site: https://code.visualstudio.com/
2. Click Download for Windows (or macOS / Linux if that's your system)
3. Open the folder that is synced with GitHub
4. Use the terminal and type code . to open the Visual studio code
5. Click File and make sure to click autosave
6. Click extensions and download Conventional Commits and Source Control
7. Create new file (Ex: helloWorld.html)
8. Use the Source Control that you downloaded to commit and push a changes to your GitHub
9. Create new file (Ex: README.md.html)
Tip
Type: git clone https://github.com/duranacristina25-byte/DSAMidterm.git
to clone your repository
Git set up
1. Go to https://git-scm.com/
2. Click Download and download Git based on your OS
3. Install Git in your device
4. To check if Git was installed correctly, open the Search bar
5. Create a folder in your desktop (ex: Durana, Cristina C.)
6. Right click to open GitBash
7. Open GitBash, make sure that you're still in the directory of your folder
8. Type:
git init
git config --global user.name "Your Name"
git config --global user.email "you@youraddress.com
git config --global push.default matching
git config --global alias.co checkout
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/duranacristina25-byte/DSAMidterm.git
git push -u origin main
9. Make sure to create a new file (Ex: README.md.html) to avoid error
10. Check if the files are already uploaded on GitHub
GitHub set up
1. Go to: https://github.com/
2. Sign up or Sign in to GitHub using your email
3. Create a repository (Ex: DSAMidterm) set it to public
4. Check if all the files from VS Code are synced
